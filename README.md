# hourglass
C++ Async Timer

## Install
```
$ git clone https://github.com/DaveyLaser/Hourglass.git
$ cd Hourglass
$ mkdir build && cd build
$ cmake ..
$ make install
```

## Preview
Run `test/main.cpp` executable for a preview of functionality.
